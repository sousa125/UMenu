# UMenu

Baixe o venv
``` 
apt install python3.10-venv
```

Crie o ambiente com o venv
```
python3 -m venv umenu
```

Ative o ambiente
```
source umenu/bin/activate
```
Atualize o pip
```
python3 -m pip install --upgrade pip
```

```
pip install -r requirements.txt
```
Execute o projeto
```
python3 manage.py runserver
```
